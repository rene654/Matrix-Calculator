# Mean Variance Standard Deviation Calculator

This project uses Python and NumPy to analyze a 3x3 matrix and calculate:

- Mean
- Variance
- Standard Deviation
- Maximum Value
- Minimum Value
- Sum

Results are returned for:
- Each column
- Each row
- The entire matrix

Built as part of the freeCodeCamp Data Analysis with Python Certification.

## Technologies Used

- Python
- NumPy

## Example

Input:

```python
calculate([0,1,2,3,4,5,6,7,8])
```

Output:

```python
{
  'mean': [[3.0, 4.0, 5.0], [1.0, 4.0, 7.0], 4.0],
  ...
}
```
